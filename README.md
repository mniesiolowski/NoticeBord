# ****NoticeBoard****

Web aplication created to publication of a notice.
Registered users can add notice 
containing title,
description, created and expires date. 
You can assign a comment to every notice.

 ****For this aplication i used :****

    IntelliJ IDEA,
    Spring Boot,
    Spring Security,
    Hibernate,
    MySQL,
    Bootstrap
